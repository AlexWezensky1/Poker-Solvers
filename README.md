# Poker-Solvers
Collaborative project to build various poker solvers, with the main goal to improve edge as players.  

##Potential features
Pre flop charts  
Node locking  
Some way to reference on phones  

##Potential games
###Flop poker  
- NLHE (already solved, used to verify)  
- PLO/8  
- PLO5 double board bomb pot  
- Ocean  
###Stud poker  
- 7 card stud  
- Razz  
###Draw poker  
- 5 card draw
